3. Create New Directives

You can invoke a directive by using:
  • Element name
  • Attribute
  • Class
  • Comment
By using Element name
By using Attribute name
By using Class name
By using Comment

Note: By setting the restrict property to "A", only the HTML element with the "w3-test-directive" attribute has invoked the directive.

The legal restrict values are:
  • E for Element name
  • A for Attribute
  • C for Class
  • M for Comment
